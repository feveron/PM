# PM — Project Management App

A simple React application created as part of a laboratory work to practice Git workflows, including version control, branching strategies, pull requests, and merge conflict resolution.

## Tech Stack

- React
- TypeScript
- Vite

## Getting Started

### Install dependencies

```bash
npm install
```

### Run the app

```bash
npm run dev
```

## Purpose

### This project demonstrates:

- repository initialization and setup
- working with branches (main, develop, feature)
- creating and reviewing pull requests
- resolving merge conflicts

## Notes

The project structure is generated using Vite and follows common frontend development practices.
